## Installation

Dependencies:
 - neovim nightly bin
 - fd
 - ripgrep
 - Python3

Install by running a single command
<br>
``` bash
bash <(curl -s https://raw.githubusercontent.com/jackm245/jvim/master/utils/installer/install.sh)
```
